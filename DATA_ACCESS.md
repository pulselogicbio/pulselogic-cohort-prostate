# Data Access -- Prostate Cancer (TCGA PRAD)

## Data Source

Expression and survival data for this cohort were obtained from UCSC Xena Browser (xenabrowser.net), which provides harmonized access to TCGA, GDC, CGCI, ICGC, and PCAWG datasets.

---

## Accessing This Data

1. Visit https://xenabrowser.net/datapages/
2. Search for cohort: **TCGA PRAD**
3. Download gene expression (STAR TPM) and survival (OS) files
4. No controlled access required for TCGA expression data
